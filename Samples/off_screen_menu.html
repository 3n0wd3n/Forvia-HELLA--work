Off Screen Menu Module

How does it works

  • We have two different key components -> Table Of Content & Content Item
  • In code they have specific naming "mod-off-screen-menu__table-of-content" & "mod-off-screen-menu__content-item"
  • If you want to use this of screen menu in site you need to take one Table Of Content and you need to create Content Items according to count of items in Table Of Content
  • Table Of Content is only once on the page and stores content items
  • Each item servers as link to anchor and anchor is each individual created Contant Item on page
  • Each Content Item has also all anchors (content items) inside iteself. And after click on it menu will pops up and you will se same items as in table of content which servers as links.
  • Part of this module is overlay that needs to be taken only once.
